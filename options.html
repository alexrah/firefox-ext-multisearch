Install as Firefox Search Engine

To use MultiSearch from the Firefox address bar or search bar, go to Firefox Preferences → Search add https://multisearch.local?q=%s and set it as your default.

To get keyword suggestions, add either https://search.brave.com/api/suggest?q=%s or https://api.qwant.com/v3/suggest?q=%s
